
####  听鱼
据说，每年德福落败的大军中，百分之八十倒在听力的战场上。

你想成为那百分之二十的幸存者吗？
你想藐视德福语速，不战而胜吗？
你想拥有同传一般的速记能力吗？

来听写吧，将你听到的句子写下来，一遍再一遍。
食堂里，地铁上，公车上，随时随地，想听就写。
一键圈出错误，不跟你拐弯抹角。
无招胜有招，有时候成为高手就这么简单。

听鱼，口袋里的听写老师。
